1. **Clone the repository**
```bash

cd ai-meme-generator
```

2. **Setup Backend**
```bash

```

3. **Configure Environment Variables**

Create `backend/.env`:
```env

```

4. **Setup Frontend**
```bash

npm install
```

5. **Start the Application**

**Terminal 1 - Backend:**
```bash
cd backend
npm start
```

**Terminal 2 - Frontend:**
```bash

npm start
```

6. **Open your browser**

## ğŸ› Known Issues & Solutions

### Memes Not Generating
**Solution:** Check that backend is running and GROQ_API_KEY is set in `.env`

### Text Overlapping
**Solution:** This should not happen with Memegen API. If it does, check backend logs.

### Slow Generation
**Solution:** Memegen API can be slow sometimes. Wait 20-30 seconds.

### "Failed to fetch" Error
**Solution:** Make sure backend is running on port 3001 before starting frontend.
